# راهنمای اضافه کردن مقالات به سیستم

این فایل راهنمای اضافه کردن 14 مقاله (7 مقاله به زبان دری و 7 مقاله به زبان پشتو) برای 7 عالم محترم به سیستم مقالات است.

## محتویات

- **7 عالم محترم:**
  1. مفتی فیض محمد عثمانی - ایمان
  2. مفتی عبدالسلام عابد - اخلاق
  3. مفتی فضل الله نوری - نماز
  4. مولانا سیدمحمد شیرزادی - اضطراب
  5. مولانا فضل الرحمن انصاری - دعا
  6. سیدعبدالاله شیرزادی - تزکیه
  7. سیدعبدالله شیرزادی - رزق

- **14 مقاله:** هر عالم یک مقاله به زبان دری و یک مقاله به زبان پشتو دارد

## فایل‌های ایجاد شده

1. **`data/articles-seed.json`** - فایل داده‌های مقالات و علما
2. **`scripts/seedArticles.js`** - اسکریپت برای import کردن مقالات به Firestore

## نحوه استفاده

### پیش‌نیازها

1. Firebase Admin SDK باید نصب باشد (به عنوان dev dependency):
   ```bash
   npm install --save-dev firebase-admin
   ```

2. کلید سرویس Firebase (Service Account Key) باید در دسترس باشد:
   - فایل `serviceAccountKey.json` در ریشه پروژه قرار دهید
   - یا متغیر محیطی `GOOGLE_APPLICATION_CREDENTIALS` را تنظیم کنید

### اجرای اسکریپت

```bash
node scripts/seedArticles.js
```

### آنچه اسکریپت انجام می‌دهد

1. **ایجاد علما:** ابتدا همه علما را در collection `scholars` ایجاد می‌کند
2. **ایجاد مقالات:** سپس همه مقالات را در collection `articles` ایجاد می‌کند
3. **محاسبه زمان خواندن:** برای هر مقاله زمان تقریبی خواندن را محاسبه می‌کند
4. **تنظیم وضعیت:** مقالات به صورت published و با تاریخ انتشار تنظیم می‌شوند

## ساختار داده‌ها

### Scholar (عالم)
```json
{
  "id": "mufti_fayz_muhammad_usmani",
  "fullName": "مفتی فیض محمد عثمانی",
  "email": "mufti.fayz@example.com",
  "bio": "بیوگرافی عالم...",
  "verified": true
}
```

### Article (مقاله)
```json
{
  "authorId": "mufti_fayz_muhammad_usmani",
  "authorName": "مفتی فیض محمد عثمانی",
  "title": "ایمان؛ ریشهٔ استقامت و آرامش",
  "language": "dari",
  "category": "iman",
  "body": "متن مقاله..."
}
```

- `salah` - نماز
- `akhlaq` - اخلاق
- `anxiety` - اضطراب
- `dua` - دعا
- `tazkiyah` - تزکیه

## نکات مهم

- مقالات به صورت خودکار published می‌شوند
- زمان خواندن بر اساس 200 کلمه در دقیقه محاسبه می‌شود
- مقالات قبلی با همان عنوان و نویسنده دوباره ایجاد نمی‌شوند
- همه مقالات با تاریخ انتشار فعلی تنظیم می‌شوند

## عیب‌یابی

اگر خطا دریافت کردید:

1. مطمئن شوید Firebase Admin SDK نصب است
2. کلید سرویس Firebase را بررسی کنید
3. اتصال به اینترنت را بررسی کنید
4. دسترسی‌های Firestore را بررسی کنید

## بعد از import

بعد از اجرای موفق اسکریپت:

- مقالات در داشبورد علما قابل مشاهده هستند
- مقالات در صفحه مقالات اصلی نمایش داده می‌شوند
- کاربران می‌توانند مقالات را بخوانند و bookmark کنند
